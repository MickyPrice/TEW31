What are we building?

In this lesson, we're going to build a simple TODO list. The list is going to be super basic, it won't even save the entries if the page is reloaded.
This should teach about creating elements, appending them and prepending them.
The page will have one input that is where the user puts in the thing they need to do and it will have a button to add this.

How do I actually make the JavaScript run on the website?

So, like the other lessons, I'm going to assume you know HTML and CSS, your HTML file will be laid out like below: <input type="text" placeholder="Todo list item here" id="input"> <button type="button" name="button" onclick="add()">Add</button> <h1>Todo List:</h1> <div id="list"> <!-- This is where the list goes --> </div> The code above will generate a screen that looks like this After this step you could choose to glam it up with CSS but for this example, In the version at the bottom of this lesson, you'll see I made a few small changes to the CSS to make it prettier, this is optional.

Adding the JavaScript

This app looks like it works but you'll find it actually doesn't do anything when it's clicked. This is because we haven't attached the JavaScript. For this example, we will use inline JS, this uses <script> tags, they work much like <style> tags except instead of holding CSS code, they hold JavaScript code.

Another way to link JavaScript is by using an external file, to do this you still use a script tag, you do it like this: <script src="linkto/file.js"></script> much like a CSS link tag, the only difference is you should load the JavaScript files just before the body tag closes, not in the head tag.
The same goes for inline js, it should go before the body tag ends.

Psudo code

This is an english outline of how you could write your code.

  • Create a function called "add()"
  • Make a variable that stores the value of the input (the input that holds the text the user has written)
  • Create an if statment that will detect if there is text inside the input.
  • if there isn't any text inside the input, alert the user that they need to fill it out
  • If there is text in the input, continue (below)
  • Create 3 variables:
  • one that creates a div - this will be the container for the todo element that will be added to the page
  • one that creates a button - this will be the "done" or "remove" button for the todo item
  • one will be a span - this will simply hold the text the user placed in the input
  • Now we'll set the innerHTML of the button to "&cross;", &cross; is an HTML entity, it looks like ✗
  • Then set the innerText of the span to the text the user put into the input
  • Now we're going to do something a little weird, we're going to set an attribute on the button that deletes the container element.
    • (If you wish to figure this out by yourself, feel free to but I'll show the code in the line below) ELEMENT.setAttribute('onclick', 'this.parentNode.remove();') Replace ELEMENT with your variable that created the button div.
  • After that, we'll append the button to the container element
  • Then, we'll append the span onto into the container
  • Finally, we'll prepend the container element onto the web page (If you copied the HTML I wrote, this will be the element with the "list" ID)
