What are we building?

We're going to build a dice roller. The roller will take a few inputs. There will be 1 input that will decide the highest value allowed to be rolled.

There will be an output section that will display the number rolled by the die.

And finally, there will be a roll dice button. This button will run the function to generate the random number.

How do I actually make the JavaScript run on the website?

So, I'm going to assume you know HTML and CSS, your HTML file will be laid out like below: <div> <input type="number" id="max" value="6" placeholder="Max dice number"> <br> <span> Your dice rolled: <span id="output"></span> </span> <br> <button onclick="roll()">Roll dice</button> </div> The code above will generate a screen that looks like this After this step you could choose to glam it up with CSS but for this example, I won't bother.

Adding the JavaScript

This little app looks like it works but you'll find it actually doesn't do anything when it's clicked. This is because we haven't attached the JavaScript. For this example, we will use inline JS, this uses <script> tags, they work much like <style> tags except instead of holding CSS code, they hold JavaScript code.

Another way to link JavaScript is by using an external file, to do this you still use a script tag, you do it like this: <script src="linkto/file.js"></script> much like a CSS link tag, the only difference is you should load the JavaScript files just before the body tag closes, not in the head tag.
The same goes for inline js, it should go before the body tag ends.

Psudo code

This is an english outline of how you could write your code.

  • Create a function called "roll()"
  • In this function, make a variable that stores the max value (set by the user in that input)
  • Then use an if statment that detects if this variable is set
  • if it isn't, do nothing (Or you can send the user a error message that tells them that they need to fill out the field)
  • If it is, create a variable that stores the randomly generated number (with a max of what the user put into the input)
  • Then display this generated number on the screen.
