What is JavaScript?

JavaScript is a computer programming language that is mostly used in web development (Although it has uses in other fields like server control, app creation etc.)
JavaScript can be used for heaps of interactability such as dropdowns, logic, automation and just handling general user interactions.

Why should I learn JavaScript?

According to seek.co.nz, there was 631 jobs involving the name JavaScript, most to do with both frontend and backend website development.
Also, according to payscale, the average pay for a web developer is $55k per year but there are higher paying jobs for people with more skills.

So from those, you can see that there's definitely availability in the job market. There's bound to be more than 631 jobs up for offer as not everyone advertises on seek. Not to mention, you can get a freelance job.
