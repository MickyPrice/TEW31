What is a statment?

Statments are basically questions (I know, that doesn't really make sense but that's how it is.)
We'll talk about just 2 statments, "IF" and "Else", these are basically questions.
There are more statments which are more useful but you'll learn those once you know the basics.
Incase you wish to expand your knowledge on this area, your next step should be looking into "For" and "While" loops.

What do these statments do?

IF statments

IF statments basically trigger only IF a value is met, for example: var age = 15; if(age =< 16) { // This code wont run } The code would not run because the =< operator means equal to or more than.
We're asking if age (the variable) is more than or equal to 16. Since age is equal to 15, the function wont continue.

Else statments

An else statment is kind of a fallback, I'll include a diagram explaining the system var age = 15; if(age =< 16) { // This would not run because }else { // This code will run in this case }

Basically, the idea of else statments is that they catch everything that does not meet the requirements for the if statment. In the example above, the If statment asks if the age is 16 or above, run the box on the left but if it's not (else), it will run the box on the right.
