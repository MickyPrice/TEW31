It's just like math!

Math in JS is just like real math believe it or not!
The hardest part is knowing the sytnax (And that's easy!).
I'm going to assume you know basic math since you're taking a course about JavaScript.

Syntax

In math, the times is written as * instead of x

Divide is written as / instead of รท

Addition is still written as +

And minus is still written as -

Lets put it into code


10 * 5; // Outputs 50 10 / 5; // Outputs 2 10 + 5; // Outputs 15 10 - 5; // Outputs 5

BEDMAS

(Brackets, Exponents, Divide, Multiply, Addition then Subtraction)
BEDMAS is the order that mathmatical opperations are performed. Math inside brackets gets done first, then the exponents, then the division and so on.

In JS, BEDMAS still applies so you'll still use round brackets.
For example, 5 + 5 * 2 will output 15
but (5 + 5) * 2 will output 20

Power

The last one which is a little strange is to the power of.
This uses a function that takes 2 paramaters. The first is the number, the second is the power of.
The function is math.pow(NUM, POWEROF). For example, say I want to find 5 to the power of 2, I would write math.pow(5, 2); // This outputs 25
