What's a variable?

A great way to describe variables is, imagine the variable as a box that you store stuff in.
You can call on these variables later on, for example, you could use a variable to store someones name.
In JavaScript, this would be written as var name = "Michael";, this variable is called name and it's value is "Michael".

What different types of variables exist?

The great part about variables is that they can store almost anything, the different types of variables are:

  • Strings (Words, wrapped in quotations eg, "This is a string")
  • Integers (Numbers)
  • Boolean (true or false)
  • Array (lists)
  • Objects (lists with keys)

What does each variable do and how do I write it?

I'll start out with the easier ones, strings, integers and booleans.

Strings

A string is just words, strings can be stuff like a person's name, a car, a country name etc.
Strings are written with quotation marks around the text, for example, if I was trying to make a variable named country with the value of New Zealand, I would write var country = "New Zealand"

Integers

Integers are numbers, for example, it could be someone's age, a year, a price etc
Integers can't have anything else but a number, for example, if you write "$100", it will be a string, not an integer.
Integers don't have quotes around them when written. If I were to make a variable called myAge and it held my age, I would write var myAge = 16

Boolean

Booleans are true or false statments, they can't be shortned to "t" or "f", they must always be true or false. You also can't wrap them in quotes or they become strings
An example of a boolean could be if someone is right handed or not, if they are, it would be true, if not, it would be false.
If I wanted to write that as a variable, I would call it isRightHanded and I'll set it true. var isRightHanded = true

Lets talk about the harder variable types. (Arrays and Objects)

Arrays

Arrays are just lists.
They can store multiple different variable types (Integer, Boolean, String, Objects and even arrays! That's right! You can have arrays inside arrays!).
So, a list is written inside square [brackets], their values are a comma seperated list, so for example, a list that contains some dinosaur names would be written like var dinosaurs = []

Objects

Objects are slightly harder. They are lists but they have keys for each values.
Objects can be used to store a whole bunch of information about a single subject, an example could be an object about a person. Below I'll show you how that would be written. var michael = { firstName: "Michael", lastName: "Price", school: "Rangiora High School", age: 16, hair: "brown", wearsGlasses: false } We probably wont use these very often during this course but it is important to know about and by all means, feel free to lean more about any of these variables elsewhere!
