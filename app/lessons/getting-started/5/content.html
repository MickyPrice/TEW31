Having all these different skills is good but how do you call them?

It's great to know all these skills but it's pretty hard to make an application without any functions to call them.
A function is basically giving a set of different functions a name that can be called on.
An example of this is, say I wanted to make a cake, in real life, I wouldn't tell the person to get flour, icing, sugar, decorations etc, I'd just say "bake a cake".
This could be considered a function.

How do functions work?

Functions are basically sections of code that get run when called.
A function has 2 parts, a name and a set of paramaters.
A function needs a name but the paramaters are optional, these are basically options for the function.
Functions are defined in JS like function name(arg1, arg2, arg3){ // This code will run }

In this function, the name of the function is "name", it has 3 arguments called "arg1", "arg2" and "arg3".
In the function, if we call one of those arguments, it will return what the user put for them.
A good example of this is a mathmatical equation, lets find the area of a square with a function.

function findArea(height, base){ var area = 0.5 * base * height; return area; } To call this function, you would write "findArea(10, 8)" (Those numbers are just random ones)

Using IF statments

In the example above, the function is great and all but what if a user forgets to put in an argument?
To solve this, we will make an if, else statment to check if they have put in both arguments. To do this, we will write the same function with some if statments.

function findArea(height, base){ // if height and base if(height && base){ var area = 0.5 * base * height; return area; }else{ return "Please fill out both arguments"; } }
